# NewSlot

NewSlotはPygameとOpenCVを使ったパチスロシミュレーターです。

---

## 特徴

- フレームレート無制限で動作します。
- 「パチスロ」としての動作を忠実に再現しています。
- ほぼ全ての要素をカスタマイズ可能です。


## 使用環境・必要ライブラリ

- 使用環境: `Python 3.13.3`
- 必要ライブラリ:  `requirements.txt` に記載


## セットアップ手順

### 1. 環境準備
```
python -m venv venv
venv\Scripts\activate
```

### 2. ライブラリのインストール
```
pip install -r requirements.txt
```


## 実行方法
```
python -m myapp.main
```
または
```
python main.py
```
